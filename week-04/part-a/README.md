# Single-Node Exercise

This exercise introduces students to solving a computational problem using a
single-node approach. The provided Jupyter Notebooks (`01-single-node.ipynb`
and `01-single-node-solution.ipynb`) contain step-by-step instructions and also
a possible solution respectively.

The exercise is based on UTD19 (https://utd19.ethz.ch/). UTD19 is a large-scale
traffic dataset from over 23541 stationary detectors on urban roads in 40
cities worldwide making it the largest multi-city traffic dataset publically
available.
